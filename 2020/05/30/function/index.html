js初级之函数

发表于 2020-05-30

函数：封装了一段可重复调用的并执行的代码块；

目的：让大量的代码重复使用（调用）

函数的使用

  1. 定义函数
    利用关键字:function 函数名 () {} ; 函数名一般是动词 get set 等
    函数表示式；var 变量名 = function(){}
  2. 调用函数： 函数名+()

    函数的参数

    利用参数可以让函数执行不同的代码 得到不同结果
  3. 形参：再函数定义时 写在（）中的数称之为形参
  4. 实参：在函数调用时（） 小括号的值称之为实参；
  5. 形参与实参时一一对应的关系；js中实参的个数未必要与形参的个数相等；定义函数的时候你可以写2个形参，但是执行的时候你可以写1个或者3个实参，甚至于不传参；因为ECMAscript 中参数在内部实际上用数组表示；函数接受的实际上是这个数组，不关系数组包含的元素；而在函数体里面实际上通过argument伪数组对象来访问这个参数数组，从而获取参数；
  6. 函数内置伪数组对象：每个函数都有一个内置的arguments伪数组对象；argument可以使用数组的length属性；并且按照索引的方式进行存储数据；但是伪数组对象没有真正数组的一些方法，如pop，push等方法；

函数的返回值;函数调用后都会返回一个结果 也就是return 的值

函数可以互相调用

js函数没有重载:java中可以为函数编写两个定义，只要这两个定义函数的签名（接受的参数的类型和数量）不同即可。但是js中函数没有签名，函数的参数是由包零个或者多个值的数组来表示的。没有函数签名，因此重载是不可能做到的
