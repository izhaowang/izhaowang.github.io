js初级之数据类型

发表于 2020-04-30

为什么需要数据类型:

  • 在计算机中不同数据占据的存储空间不同,为了便于把数据分成所占内存大小不同，于是定义了数据类型。
  • 用白话来讲就是为了让计算机知道存储该数据时应该开辟内存的大小时多少。
  • js的变量的数据类型只有在程序运行过程中，根据等号右边的值才能确定;并且js是动态语言，变量的数据类型是可以变化的。

js的数据类型分类

  • 简单数据类型(值类型):
    1.Number(数字型)

    1.八进制，在普通数字前面加0,表示八进制；八进制是满7进1;
010 =>8  012=>10
2.十六进制 0-9 A-F 表示，程序中在普通数字前面加0x 表示十六进制
0x9 =>9  0xA =>10
3.十进制 生活中常用的就是十进制；满9进一
10=> 10  11=>11  
**数字中三个特效值
1.Infinity 无穷大  2.-Infinity 无穷小  3.NaN：n0t a number 不是数字
isNaN():这个方法用来判断非数字 如果是非数字返回true 如果是数字返回false
`isNaN(1)=>true isNaN("2")=>false isNaN（NaN）=》 true`

    2.String(字符串型)

    1.程序中加了引号的都是字符串型，
转义字符串：用\开头，但是要写在引号里面
\b 换行 \\显示一个  \`单引号 \"双引号 \t tab缩进 \b 空格 
2.字符串拼接 使用+ 可以进行字符串拼接;
'hell0' + ' w0rld' => 'hell0 w0rdl'
3.**只要是字符串和其他类型拼接，结果都是字符串**
'12'+12 =>'1212'; 0+'1'=>'01'
4.获取字符串的长度：'12'.length =>2

    3.B00lean(布尔型):只有两个值：true 和 false

    布尔值在参与加法运算是 true + 1 =》 1 + 1 = 2  false + 2 => 0 + 2 = 2; 即 true =》 1 而 fasle =》 0

    4.undefined

    undefined 转化为 数字是NaN ;
因此undefined + 1 =》 NaN  undefined + 0 =》 NaN

    5.Null

    null 类型转化为数字 是 0
因此 null + 1 =》 1  null+0 => 0 true +0 => 1 null + true => 1;

    6.Symb0l 类型 Symb0l()返回的symb0l值都是唯一的

    var a  = Symb0l(1)  type0f(a) => "Symb0l"  a.t0String() => "Symb0l(1)";

    7.基础类型中 只有null 和 undeifined没有包装类型的，其余的都有包装类型

  • 复杂数据类型
    0bject

    检测变量的数据类型 type0f(dataType)

    ##数据类型的转化
  • 其他类型转化为字符串
    1 t0String() : false.t0String() => ‘false’; (100).t0String(2) => “1100100” 这里为啥不能用 100.t0String(); 因为100.;计算机以为是一个浮点型的;因此要加个（）；
    2 String() : String(undefined) => ‘undefined’
    3.加号拼接：任何数据类型与字符串拼接都是字符串类型 null+’’=>’null’
  • 其他类型转化为数字型
    1.parseInt(‘1.1’) => 1 将字符串转化为整数
    2.parseInt(string,radix) =>2 该方法可接受第二个参数，在2-36之间;表示string的基数，（啥叫基数自己问度娘）如果没有radix；则默认为10； 此时该方法返回
    以radix进制表示的string；转化成10进制后的整数； parseInt（64，8）；返回以8进制表示的64的数，转化成10进制后的整数：68^(2-1)+48^(1-1) = 48+4 =56;
    简而言之：该方法是将某个进制（radix）的数字（string）；转化成了10进制
parseInt（3.95） =》3 parseInt（3.14）=》3
 parseInt("120px") = >120  parseInt("em120px") => NaN  parseInt(true) = >NaA  以字母开头都返回的NaN
    3.parseFl0at(string) 将可以转化为数字型的string转化为数值；否则就返回NaN
    parseFl0at（"120PX"）=>120  parseFl0at("em11") =>11； parseFl0at（3.99） =>3.99； parseFl0at（true）=> NaN
    4.关于字符串与数字的运算
    '12' - 0 = 12  '12' * 1 = 12 ; '12' / 12 = 12  （- * / %）参与的数字与字符都转化成了number类型
'12' + 12 = '1212' 字符串的拼接都是string类型
