javascipt初级之js介绍

javascript 由布兰登·艾奇(1961年~ )发布;在王景公司最初命名为livescript;后来在sun公司改名为javascript;

javascript 特点

javascript 是运行在客户端的语言,也可以基于node.js技术服务器端来编程；

javacript（以下写为js） 作用

最初只是用来进行表单验证；
后来随着技术发展,js可以用于网页特效，服务端开发node.js，桌面程序electrol；App(cordoval);控制硬件ruff；游戏开发等;

浏览器执行js简介

浏览器

  • 渲染引擎:解析HTML,CSS 俗称内核
  • js解释器:读取js并运行如chrome（V8）
  • 浏览器通过内置的js解释器来执行js代码。js解释器执行代码是逐行解释每一句（转化为机器语言二进制）；然后计算机去执行，所以js语言归为脚本语言

js组成

  1. ECMAScript：js的语法（规范）
  2. DOM 文档对象模型
  3. BOM 浏览器对象模型

js书写的位置

  1. 行内 写在元素内部
  2. 内嵌 卸载script 标签中
  3. 外部 利用script 标签的src 属性引入

为什么需要数据类型:

  • 在计算机中不同数据占据的存储空间不同,为了便于把数据分成所占内存大小不同，于是定义了数据类型。
  • 用白话来讲就是为了让计算机知道存储该数据时应该开辟内存的大小时多少。
  • js的变量的数据类型只有在程序运行过程中，根据等号右边的值才能确定;并且js是动态语言，变量的数据类型是可以变化的。

语言

发表于 2020-04-30

计算机不能直接了解任何机器语言之外的语言，因此需将编程通过翻译器生成计算机可以识别的机器语言（二进制）。

翻译的方式有两种：

  • 一个是编译： 就是再执行代码之前进行编译，生成中间代码文件；然后再执行中间文件；java 属性编译性语言，先编译生成中间文件，然后执行
  • 另一种是解释: 再运行代码的时候及时解释，并且立即执行； js属性解释性语言，解释后立即执行。

标识符：指开发人员为变量，属性，函数取得名字

关键字：js本身已经使用的词，用var ，let ，function等；不允许充当变量，方法名；

保留字: 预留的关键字，现在不用，可能以后版本更新会用到；

js变量

变量的定义

  • 变量是程序在内存中申请一块来存放数据的空间;你可以任务变量是一个盒子,里面存放这数据;我们可以通过变量名来找到存放的数据。

    变量的使用（分为两步）

  1. 声明变量
    • 声明变量用的关键字var 例如: var age 声明了一个变量名为age的变量; 此时也在内存中开辟了一块空间,用来存放变量数据
  2. 赋值
    • 例如age = 10; 将10 赋值给了age; 此时这块空间存放的数据为10;

变量的初始化

  • 其实就是变量声明并赋值 例如var myName = "SpongBob"; 声明了一个变量名为myName的变量，并且该变量存放的数据是“SpongBob”;

  • 可以同时声明多个变量

    多个变量名之间用英文输入法下的 , 隔开，最后一个变量名需用 ;

注意：

  • 1 var sex ===> undefined 只声明，不进行赋值是undefined
    2 console.log(sayHello) 报错， 不进行声明直接使用变量会报错；
    3 sayHello = "你好啊" 变量可以不声明直接使用; ===> “你好啊;
  • 1 变量名不能以数字开头 var 12age 这种变量名12age是错误的；
    2 变量名有大小写的区别 var name 和 var Name 这两个是不同的变量;
    3 变量名不能是关键字或者保留字 var var 你不能用var关键字来充当变量名;
    4 变量名要有意思 你不能使用var abc用abc 这种无意思的变量名来报错数据；
    5 关于符号命名，只允许”$” 和 “_”;
